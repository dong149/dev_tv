## 😢 Problem

   <p align="center"><img src="https://github.com/dong149/dev_tv/blob/master/why.png" width="600%"/></p>

개발 관련한 영상들을 많이 찾아보던 중, fow.kr이라는 서비스와 연관지어 개발 관련 영상을 보여주는 사이트가 있으면 어떨까 생각이 들었습니다.

유튜브 영상을 복사 붙여넣기하면 바로 영상이 등록되는 fow.kr 의 UX에서 아이디어를 얻어서 진행한 프로젝트입니다.

### 유저의 반응은 차가웠습니다 😢

이 서비스는 후에 롤링페이퍼 서비스를 개발할 때, SEO에 집착하게 된 시작 포인트가 되게 됩니다.
어떻게 하면, 사용자가 한 번이라도 볼 수 있을까, 검색창에 잘 검색이 될 수 있을까에 대한 고민을 하게 되었습니다.

### 이후에 토이 프로젝트로 제대로 만들어보고 싶습니다 😀

결과는 슬펐지만, 많은 것을 느끼게 해준 프로젝트였습니다. 이후에는 더욱 체계적인 설계와 계획을 가지고 제대로 개발해보겠습니다!
